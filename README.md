# 邮件工作总结汇总程序

一个强大的Python工具，用于自动处理邮件文件并生成格式化的工作总结报告。

## ✨ 功能特性

- 📧 **智能邮件解析**: 自动解析.eml格式邮件文件
- 📅 **日期提取**: 从文件名和邮件头智能提取日期信息
- 🔄 **重复文件处理**: 自动检测并合并同一日期的重复邮件
- 📊 **月度报告生成**: 按月份组织并生成Markdown格式报告
- 🎯 **灵活月份选择**: 支持生成指定月份的报告
- 🌐 **多编码支持**: 自动检测并处理多种字符编码
- 📈 **统计信息**: 提供详细的处理统计和进度信息

## 🚀 快速开始

### 安装依赖

```bash
pip install -r requirements.txt -i https://mirrors.huaweicloud.com/repository/pypi/simple
```

### 基本使用

```bash
# 交互式选择月份（默认行为）
python main.py

# 只显示统计信息
python main.py --stats

# 生成指定月份的报告
python main.py --months 2024-07,2024-08

# 生成所有月份的报告
python main.py --months all

# 指定邮件目录和输出目录
python main.py --dir ./邮件文件夹 --output ./报告输出
```

## 📋 命令行参数

| 参数 | 简写 | 说明 | 默认值 |
|------|------|------|--------|
| `--dir` | `-d` | 邮件文件目录 | `./工作总结` |
| `--output` | `-o` | 输出目录 | `./output` |
| `--months` | `-m` | 指定月份 (格式: 2024-07,2024-08) | `交互式选择` |
| `--stats` | `-s` | 只显示统计信息 | - |
| `--verbose` | `-v` | 详细日志输出 | - |
| `--version` | | 显示版本信息 | - |

## 📧 支持的邮件格式

- ✅ .eml格式邮件文件
- ✅ 多种字符编码 (UTF-8, GBK, GB2312等)
- ✅ 多部分邮件 (multipart)
- ✅ 中文邮件内容

## 📝 生成的报告

程序会生成以下类型的报告：

1. **月度报告**: `YYYY年MM月工作总结.md`
   - 按日期组织的工作内容
   - 包含统计信息和元数据
   - Markdown格式，便于查看和分享

2. **汇总报告**: `工作总结汇总报告.md`
   - 整体统计信息
   - 时间范围和月份分布
   - 处理结果概览

## ⚙️ 配置说明

可以通过修改 `config.py` 文件来自定义程序行为：

```python
# 邮件文件目录
WORK_SUMMARY_DIR = Path("./工作总结")

# 输出目录
OUTPUT_DIR = Path("./output")

# 支持的编码
DEFAULT_ENCODING = "utf-8"
FALLBACK_ENCODINGS = ["gbk", "gb2312", "utf-16"]

# 内容标记
CONTENT_START_MARKERS = ["工作内容", "今日工作", "工作总结"]
CONTENT_END_MARKERS = ["发送时间", "发件人", "---"]
```

## 🔧 高级功能

### 重复文件处理

程序自动检测同一日期的多个邮件文件，并智能合并：
- 选择内容最完整的邮件作为基础
- 合并不同的内容部分
- 保留所有文件名信息

### 日期提取

支持多种日期格式：
- 文件名中的日期: `[2024-07-15]`, `[2024-7-15]`
- 邮件头中的Date字段
- 自动验证工作日有效性

### 编码处理

智能处理多种字符编码：
1. 自动检测文件编码
2. 尝试多种备用编码
3. 优雅降级处理编码错误

## 🐛 故障排除

### 常见问题

1. **编码错误**
   - 确保邮件文件编码正确
   - 检查 `config.py` 中的编码设置

2. **日期解析失败**
   - 检查文件名格式是否符合要求
   - 确认邮件头包含有效的Date字段

3. **内存不足**
   - 对于大量邮件文件，考虑分批处理
   - 增加系统内存或使用更强大的机器

### 日志文件

程序会生成 `email_processor.log` 日志文件，包含详细的处理信息和错误记录。
